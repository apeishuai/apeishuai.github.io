身体几大系统

Tags: health

🧬 人体健康诊断全景地图（器官归属系统 + 对应检查指标）

系统 功能 核心器官/组织 主要检查指标
心血管系统 运输血液 / 氧气 / 养分 ❤️ 心脏（五大核心脏器之一） 心电图（心律失常 / 心肌缺血）心脏彩超（瓣膜 / 心腔大小 / 射血分数 EF > 50%）心肌酶谱（肌钙蛋白 < 0.04 ng/mL，心肌损伤标志物）
🩸 血管（动脉 / 静脉） 颈动脉彩超（斑块 / 狭窄）血脂四项（LDL-C < 3.37 mmol/L，评估动脉硬化风险）
呼吸系统 吸入氧气 / 排出二氧化碳 🫁 肺（五大核心脏器之一） 低剂量螺旋CT（肺结节 / 肺癌筛查，金标准）胸部X线（基础筛查，肺炎 / 结核）肺功能检测（FEV1/FVC > 70%，慢阻肺筛查）血氧饱和度（SpO₂ > 95%）
消化系统 食物摄入 / 消化 / 吸收 / 排泄 🧫 肝脏（五大核心脏器之一） 腹部B超（脂肪肝 / 囊肿 / 占位）肝功能全套：ALT（男 < 50 / 女 < 40 U/L）、AST、胆红素（< 21 μmol/L）肝脏弹性检测（肝硬化程度）
🫘 脾脏（五大核心脏器之一） 腹部B超（脾脏厚度 < 4.0cm，长径 < 12cm）血常规（血小板 100-300×10⁹/L，脾亢时降低）
🥘 胆囊 / 胰腺 腹部B超（胆结石 / 息肉 / 胰腺囊肿）血淀粉酶（胰腺炎指标）空腹血糖（胰腺内分泌功能，< 6.1 mmol/L）
🫄 胃 / 肠 胃镜（食管 / 胃 / 十二指肠，金标准）肠镜（结直肠，金标准）C13 / C14呼气试验（幽门螺杆菌）粪便潜血（消化道出血 / 肿瘤线索）
泌尿系统 过滤血液 / 生成尿液 / 排出代谢废物 🧻 肾脏（五大核心脏器之一） 肾功能：血肌酐（男 57-97 / 女 41-73 μmol/L）血尿酸（男 < 428 / 女 < 357 μmol/L）尿常规（尿蛋白阴性 / -，潜血）肾脏B超（结石 / 囊肿 / 大小 / 形态）
🫗 膀胱 / 输尿管 泌尿系B超（结石 / 肿瘤）
内分泌系统 分泌激素 / 调节全身代谢 🧬 甲状腺 甲状腺B超（结节 / 大小 / 血流）甲状腺功能（T3 / T4 / TSH，甲亢 / 甲减）
🩸 胰岛（位于胰腺内） 空腹血糖（< 6.1 mmol/L，≥ 7.0 诊断糖尿病）糖化血红蛋白（HbA1c < 7%，反映近3个月平均血糖）
生殖系统 生殖繁衍 🧬 前列腺（男） 前列腺B超（增生 / 钙化）血清PSA（前列腺癌标志物，> 4 ng/mL 需警惕）
🫃 子宫 / 卵巢（女） 妇科B超（肌瘤 / 囊肿 / 内膜厚度）宫颈TCT（细胞学，宫颈癌筛查）HPV检测（高危型病毒）
🫂 乳腺（女） 乳腺彩超（结节 / 增生）乳腺钼靶（40岁以上，乳腺癌筛查）
血液 / 免疫系统 运输免疫细胞 / 凝血 / 抵御感染 🩸 全血细胞（由骨髓生成） 血常规：白细胞（4.0-10.0×10⁹/L）、红细胞、血红蛋白、血小板
🫘 脾脏（同时是免疫器官） （参见消化系统-脾脏）
骨骼 / 肌肉系统 支撑身体 / 保护脏器 / 运动 🦴 骨骼 / 关节 骨密度检测DXA（骨质疏松 T值 ≤ -2.5）X线 / CT / MRI（骨折 / 椎间盘突出 / 关节炎）
神经系统 接收传导信号 / 控制全身 🧠 大脑 / 脑血管 头颅CT / MRI（脑梗 / 肿瘤 / 出血）经颅多普勒TCD（脑血管血流速度）颈动脉彩超（颈动脉斑块 → 推测脑血管风险）

📊 慢性病类型 vs 主要危险因素

慢性病类型（代表性疾病） 不可控因素 行为因素 代谢因素 其他因素
心血管疾病（高血压、冠心病、脑卒中） 遗传、年龄增长 高盐（日均10克）、吸烟、酗酒、少动 肥胖/超重、高血脂、高血糖 精神紧张、压力大
癌症（肺癌、肝癌、胃癌等） 遗传 吸烟（肺癌主因）、高盐/腌制饮食、酗酒、少动 肥胖/超重 空气污染、职业有害物、慢性感染（幽门螺杆菌、HPV、乙肝/丙肝）
慢性呼吸系统疾病（COPD、慢支、哮喘） — 吸烟（最重要）、长期有害气体/颗粒暴露 — 空气污染（PM2.5、油烟/燃煤）、职业粉尘、反复感染
糖尿病（2型糖尿病） 遗传、家族史 高糖高脂饮食、能量过剩、少动、久坐 肥胖/超重（成人超重率50.7%） —
